# Projetos Iniciais em Python

Este repositório reúne meus primeiros projetos desenvolvidos em Python durante meus estudos de lógica de programação pelo Curso em Vídeo (Professor Gustavo Guanabara) e práticas pessoais.

Os códigos aqui representam o início da minha jornada como desenvolvedora e mostram minha evolução nos fundamentos da linguagem Python.

## O que você vai encontrar aqui

Projetos focados em:
- Entrada e saída de dados
- Variáveis
- Operações matemáticas
- Condições (if / else)
- Laços de repetição
- Lógica de programação
- Formatação de resultados

## Conteúdos

Este repositório contém diversos pequenos programas, como por exemplo:
- Tabuada interativa  
- Conversor de reais para dólares  
- Conversor de Celsius para Fahrenheit  
- Cálculo de área de uma parede e quantidade de tinta  
- Outros exercícios e projetos de estudo  

## Objetivo

Este repositório tem como objetivo registrar minha evolução como desenvolvedora, organizar meus estudos e servir como base para projetos mais avançados no futuro.
